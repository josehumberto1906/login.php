# Sistema de Login em PHP

Projeto simples de autenticação de usuário desenvolvido para estudo de programação web.

## Funcionalidades

- Formulário de login com CPF e senha
- Validação de dados
- Controle de sessão com PHP
- Redirecionamento para página principal após login

## Tecnologias utilizadas

- HTML
- PHP
- Apache (XAMPP)

## Estrutura do projeto

index.php → página de login  
login.php → validação do usuário  
principal.php → página principal após login

## Objetivo do projeto

Projeto criado para praticar conceitos básicos de desenvolvimento web utilizando PHP.
